Detailed Description

Colors the shape using the coordinates of the vertices as indicators for the rgb value.

The entries are the coordinates of each vertex but normalized to values between 0 and 255.

Author
Emanuel Laude and Zorah Lähner

Member Function Documentation

void CoordinateColoring::calculateColors ( )
private

Calculates the rgb color for each vertex.

Calculates the min and max coordinate values on each axis by iterating over all vertices. Then assigns each vertex their x- y- and z- coordinate as r- g- and b-value normalized on the interval 0 to 255.

vtkSmartPointer<vtkUnsignedCharArray> CoordinateColoring::getColors ( )
inline

Returns an array with the rgb values for each vertex.

The array has 3 columns and as many columns as the shape has vertices.
