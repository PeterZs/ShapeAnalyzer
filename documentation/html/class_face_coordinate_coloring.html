Detailed Description

Colors the shape using the coordinates of the faces as indicators for the rgb value.

The entries are the interpolated values of the vertices of each face but normalized to values between 0 and 255. The class inherits from CoordinateColoring the only overwritten function is the calculation of the color for each face.
