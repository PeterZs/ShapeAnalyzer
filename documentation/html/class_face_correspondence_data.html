Detailed Description

Stores multi correspondences between faces of an arbitrary number of shapes.

The main implementation can be seen in CorrespondenceData, this class purely insures that point and face correspondences can not be mixed up in some data structures (especially faceCorrespondencesData_ in ShapeAnalyzer).
