vtkSmartPointer<vtkDoubleArray> ScalarFaceAttribute::getScalars ( )
inline

Returns the vtkDoubleArray containing the attributes.

The size of the returned array should be equal to the number face of the shape. The order of the values normally corresponds to the one in the vtkPolyData structure of the shape.
